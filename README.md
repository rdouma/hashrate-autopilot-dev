# Hashrate Autopilot - dev store

Private Umbrel community app store for testing **pre-release** builds of
[rdouma/hashrate-autopilot](https://github.com/rdouma/hashrate-autopilot)
before they ship to the production store.

Add to Umbrel: **Settings -> App Stores -> Add** ->
`https://github.com/rdouma/hashrate-autopilot-dev`

Then install **Hashrate Autopilot (dev)**. It installs alongside the
production app (distinct store id `rdouma-dev`), uses its own data
directory, and pins a pre-release image tag. Not for end users.
